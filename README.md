# Capybara Masbro 

## Prerequisites
- Java
- Maven, to install refer to : `https://phoenixnap.com/kb/install-maven-windows`

## How to Run
#### Build the jar file
```bash
mvn clean install
```

#### Run the jar file
```bash
java -jar target/CBR-1.0.jar
```

## Adding Dependencies
1. Go to `pom.xml`
2. Add new dependecy under `<!-- Add Another Dependencies Here -->` tag
